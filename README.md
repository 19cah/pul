## Directory Structure

```
.
├── mobile // contains exponent project
│   ├── assets
│   │   ├── fonts
│   │   ├── html
│   │   └── images
│   ├── components
│   ├── config
│   ├── navigation
│   ├── screens
│   ├── stores // MobX stuff
│   └── utils
├── server // contains worker scripts on Heroku
└── template-files // example files for api keys needed
```

- Exponent
- React Native
- ex-navigation
- MobX
- Firebase
