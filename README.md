## Directory Structure

```
.
├── mobile // contains exponent project
│   ├── assets
│   │   ├── fonts
│   │   ├── html
│   │   └── images
│   ├── components
│   ├── config
│   ├── navigation
│   ├── screens
│   ├── stores // MobX stuff
│   └── utils
├── server // contains worker scripts on Heroku

```

- Exponent
- React Native
- ex-navigation
- MobX
- Moment.js
